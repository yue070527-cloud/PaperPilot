# 第二阶段：完整版提交（4周）

## 目标
功能完备，可打包成单文件桌面应用（exe/dmg），体积<60MB。

## 技术任务

### 1. 多维打分与可视化
- 五维打分：相似度、关键词、方法、结论、时效
- Flet 嵌入 Matplotlib 雷达图（或 ASCII 雷达图过渡）
- 展示历史均值对比线

### 2. Cross-Encoder 重排序
- 接入 ms-marco-MiniLM-L-6-v2（~80MB，CPU实时）
- 流程：FAISS 召回 Top100 → Cross-Encoder 精排 → Top20

### 3. AI 解释生成
- Top5 调用 DeepSeek Chat 生成结构化分析（契合原因、方法、结论、局限、展望）
- 反事实解释："若采用XX数据集，契合度可从0.72提升至0.89"

### 4. 阅读管理
- 三级状态：未读 / 略读 / 精读
- 星标收藏、AI笔记（方法/结论/局限）、用户批注
- 笔记内容向量化，反哺推荐

### 5. 自适应权重
- 记录收藏/跳过/精读到 feedback 表
- EMA 动态调整权重，UI 实时展示变化

### 6. 系统托盘与推送
- plyer/PyQt QSystemTrayIcon 跨平台托盘
- 右键菜单：立即刷新推荐 / 打开主窗口 / 推送设置 / 退出
- **推送周期配置**：用户在设置页自定义推送频率（每日 / 每周 / 自定义天数），支持不同课题设不同周期
- **后台定时任务**：APScheduler 或 threading.Timer 按预设周期触发，后台静默抓取新论文
- **三种推送模式**：
  - **科研快讯**：定时查询 arXiv API（按日期范围）及 OpenAlex（按新收录时间），KeyBERT + 向量相似度快速匹配课题，匹配度超阈值则纳入推送列表；推送卡片含标题、第一作者、一句话摘要
  - **课题全量重排**：对课题文献池重新跑 FAISS + Cross-Encoder 排序，与上次排序对比，输出"新上榜 / 排名上升 / 排名下降"变动报告
  - **Digest 周报**：每周定时生成，汇总本周新增文献数、阅读完成数、收藏数、Top 5 推荐，输出 Markdown 格式摘要
- 桌面弹窗：Windows toast / macOS notification，展示"本周新增 N 篇相关论文"，列出 Top 3 标题，点击跳转应用内详情
- **应用内通知中心**：侧边栏通知入口，按时间线展示推送历史，支持按课题筛选、标记已读/未读

### 7. 导出功能
- Markdown 对比表格、CSV 导出
- 综述初稿：拼接 Top 论文 AI 分析输出结构化草稿

### 8. UI 完善与打包
- 课题管理、推送历史、设置页（API Key/权重滑块/数据源开关）
- PyInstaller --onefile --windowed + UPX，目标 <60MB

## 演示亮点

- 收藏后权重实时变化，断网秒切离线模式
