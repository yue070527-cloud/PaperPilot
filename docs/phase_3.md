# 第三阶段：进阶深化版（8周）

## 目标
技术深度与展示效果拉满，加入中文全数据源，冲击高奖项。

## 技术任务

### 1. 交互式知识图谱
- PyQt6 + QWebEngineView 嵌入 D3.js，节点为概念/方法/作者/数据集
- 隐藏连接发现：跨领域共同技术路线自动标红
- Gap Detection：课题涉及但文献未覆盖的概念组合红色高亮
- 导出：PNG/SVG/Markdown/Obsidian Canvas

### 2. PDF 章节级分析
- PyMuPDF 识别 Abstract/Methods/Results/Conclusion，分章节送 AI 生成摘要
- 扫描版 PDF 自动跳过并提示用户

### 3. 中文数据源
- 知网/万方导出文件（EndNote/RefWorks）批量解析，中文 KeyBERT 适配
- 中英文混合向量索引

### 4. 微信推送（彩蛋）
- 接入 PushPlus/Server 酱，非核心功能，仅推送标题和链接

### 5. 多课题管理
- 独立文献池，一键切换，课题间覆盖度对比，本地配置文件数据隔离

### 6. 性能优化
- FAISS IndexIVFFlat 大数据量加速，异步线程池 + 指数退避重试
- 定期清理 30 天前未标记低分论文

### 7. 离线模型完善
- Ollama 本地 Embedding（nomic-embed-text）+ 文本生成（qwen2.5/phi3）
- 离线模式完整支持排序、笔记、图谱浏览

### 8. 展示效果强化
- 知识盲区实时标红动画，自适应权重变化曲线
- 课题演化追踪：课题描述修改后自动重新评估历史文献库

## 演示亮点
- 动态知识图谱缩放/拖拽/节点跳转，"隐藏连接"展示跨领域技术关联
- 知识盲区一键生成"潜在创新方向"报告

## 交付物
- 完整进阶版桌面应用
- 答辩 PPT（算法流程图、架构图、成本-效果曲线）
- 3 个真实课题测试报告（准确率、覆盖度分析）
- 技术论文/设计文档（可选，冲击特等奖加分）
