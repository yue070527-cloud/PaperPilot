# PaperPilot —— 面向课题攻关的可解释智能文献工作流系统

## 定位
可解释、课题驱动、本地离线的桌面端文献智能助手，解决"导师让读100篇文献"的筛选痛点。

## 技术栈
Python + Flet + FAISS + SQLAlchemy + DeepSeek API + Ollama

## 核心功能

### 1. 文献检索
- 英文：arXiv + OpenAlex（免Key）
- 中文：知网/万方导出文件批量导入 + 本地PDF文件夹导入
- 元数据：Crossref、DBLP
- 检索：KeyBERT自动提取技术术语，支持手动关键词

### 2. 三层契合度打分
- **粗排**：FAISS向量相似度 + KeyBERT术语共现 + 指数时间衰减 → Top100
- **精排**：Cross-Encoder（ms-marco-MiniLM）重排序 → Top20
- **可解释打分**：自适应权重（相似度/关键词/方法/结论/时效）线性加权，雷达图展示 + 反事实解释
- 仅Top5调用DeepSeek Chat生成结构化分析（方法/结论/局限/展望）
- **自适应学习**：记录收藏/跳过/精读行为，EMA动态调整权重

### 3. 阅读工作流
- 阅读预算：设定每日时长/篇数，智能选最优批次
- 三级状态：未读 / 略读（AI摘要） / 精读（全文+笔记）
- AI笔记：自动生成结构化笔记，用户可批注；笔记入向量库反哺推荐
- 知识覆盖雷达图：标出子领域"知识盲区"

### 4. 交互式知识图谱
- 节点：概念/方法/作者/数据集，边为共现关系
- 隐藏连接发现：跨领域共同技术路线
- Gap Detection：红色标出课题涉及但文献未覆盖的概念组合
- 导出：PNG/SVG/Markdown/Obsidian Canvas

### 5. 智能推送
- 系统托盘驻留，右键"立即刷新推荐"
- **推送周期可配置**：用户可自定义推送频率（每日 / 每周 / 自定义天数），不同课题可设不同周期
- **三种推送模式**：
  - **科研快讯**：定时抓取 arXiv 最新 preprint 及已发表新刊，按课题匹配度筛选，推送标题 + 一句话摘要，像"课题专属的 RSS 阅读器"
  - **课题攻关全量排序**：对当前课题下所有已收录文献重新打分排序，输出变动报告
  - **Digest 周报**：汇总本周新增文献 + 阅读统计（读了/跳过了几篇）+ Top 推荐，生成邮件风格摘要
- 桌面弹窗展示新文献数量及 Top 3 标题，点击跳转应用内详情
- 应用内通知中心：按时间线展示所有推送历史，支持按课题筛选
- 后期可选接入微信推送（PushPlus/Server 酱，Phase 3 彩蛋）

### 6. 知识整理导出
- 多论文对比表格（方法/结论/局限/数据集），Markdown/CSV导出
- 综述初稿：拼接Top论文AI分析，输出结构化草稿
- 笔记汇总：AI笔记与用户批注统一导出

## 技术架构
- **数据层**：SQLAlchemy ORM + SQLite + FAISS向量索引 + diskcache API缓存
- **算法层**：DeepSeek Embedding（在线）/ MiniLM+Ollama（离线）+ KeyBERT + Cross-Encoder + EMA自适应权重
- **应用层**：Flet GUI（中期/完整版），第三阶段图谱模块可切PyQt6+QWebEngineView
- **打包**：PyInstaller --onefile --windowed + UPX，目标<60MB
- **离线兜底**：无网络自动加载本地缓存+Ollama，界面显示"离线模式"

## 分阶段安排
- **第一阶段（2周）**：终端闭环 + Flet基础UI + 50篇测试数据 + 中期报告录屏
- **第二阶段（4周）**：多维打分+雷达图 + Cross-Encoder + 阅读管理 + 托盘推送 + 打包分发
- **第三阶段（8周）**：D3.js知识图谱 + PDF章节级分析 + 中文数据源 + 多课题管理 + 性能优化

## 风险控制
- API不稳：自动重试+备用API（通义千问/智谱），现场秒切
- 网络中断：预缓存50篇+Ollama离线演示
- 评委质疑"AI套壳"：准备算法伪代码，现场推导四层架构
- 学术诚信：界面标注"AI辅助需人工核实"
