# 第二阶段：完整版提交（4周）

## 目标
功能完备，可打包成单文件桌面应用（exe/dmg），体积<60MB。

## 技术任务

### 1. 多维打分与可视化
- 五维打分：相似度、关键词、方法、结论、时效
- Flet 嵌入 Matplotlib 雷达图（或 ASCII 雷达图过渡）
- 展示历史均值对比线

### 2. Cross-Encoder 重排序
- 接入 ms-marco-MiniLM-L-6-v2（~80MB，CPU实时）
- 流程：FAISS 召回 Top100 → Cross-Encoder 精排 → Top20

### 3. AI 解释生成
- Top5 调用 DeepSeek Chat 生成结构化分析（契合原因、方法、结论、局限、展望）
- 反事实解释："若采用XX数据集，契合度可从0.72提升至0.89"

### 4. 阅读管理
- 三级状态：未读 / 略读 / 精读
- 星标收藏、AI笔记（方法/结论/局限）、用户批注
- 笔记内容向量化，反哺推荐

### 5. 自适应权重
- 记录收藏/跳过/精读到 feedback 表
- EMA 动态调整权重，UI 实时展示变化

### 6. 系统托盘与推送
- plyer/PyQt QSystemTrayIcon 跨平台托盘
- 右键菜单：立即刷新推荐 / 打开主窗口 / 退出
- 桌面弹窗展示 Top N

### 7. 导出功能
- Markdown 对比表格、CSV 导出
- 综述初稿：拼接 Top 论文 AI 分析输出结构化草稿

### 8. UI 完善与打包
- 课题管理、推送历史、设置页（API Key/权重滑块/数据源开关）
- PyInstaller --onefile --windowed + UPX，目标 <60MB

## 演示亮点
- 完整流程：输入 → 抓取 → 重排序 → 雷达图 → AI解释
- 收藏后权重实时变化，断网秒切离线模式
